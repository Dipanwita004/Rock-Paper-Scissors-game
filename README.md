# 🪨 Rock Paper Scissors Game

## 📌 Description

This is a simple and interactive Rock Paper Scissors game where the user plays against the computer. The game uses core JavaScript logic to determine the winner and updates the score dynamically.

## 🚀 Features

* Random computer choice generation
* Real-time winner detection logic
* Score tracking (User vs Computer)
* Instant result display (Win/Lose/Draw)
* Dynamic UI updates with colors and messages

## 🛠️ Technologies Used

* HTML
* CSS
* JavaScript (Vanilla JS)

## ▶️ How to Run

1. Clone the repository
2. Open the project folder
3. Run the file:
   Open index.html in your browser

## 📷 Example

* Click on Rock / Paper / Scissors
* Computer generates its choice
* Result is displayed instantly
* Score updates automatically

Example Output:
You Win!!! Your rock beats scissors
Score → User: 1 | Computer: 0

## 🧠 Core Logic Used

* Random selection using Math.random()
* Conditional logic to decide winner
* Event listeners for user interaction
* DOM manipulation for updating UI and scores
